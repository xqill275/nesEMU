**Current limitations**
- Mapper 0 (NROM) only
- No audio (APU not implemented)

### Status Legend
- **Playable** – Fully playable from start to finish
- **In-Game** – Runs with noticeable issues
- **Boots** – Reaches gameplay but is broken
- **Does Not Boot** – Fails to start
